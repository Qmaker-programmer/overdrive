# 🏎️ Overdrive!

¡Bienvenido a **Overdrive!**, un juego de acción y velocidad desarrollado en HTML5 Canvas! 

## 🎮 Sobre el juego
**Overdrive!** es un título de supervivencia y físicas donde controlas a un personaje a través de diferentes biomas generados dinámicamente. Enfréntate a desafíos, vigila tu velocímetro y sobrevive el mayor tiempo posible

Todo el juego se ejecuta directamente en el navegador sin necesidad de instalar dependencias pesadas ni motores gráficos de terceros.

Nota: Guardado que usa Cookies del navegador, en menu de pausa

## 🛠️ Tecnologías utilizadas
* **HTML5 Canvas:** Renderizado del juego y la interfaz (HUD).
* **JavaScript (Vanilla):** Lógica del juego, físicas, cálculo de colisiones y manejo de cámara.
* **CSS3:** Estilizado del contenedor y tipografía personalizada (*Bungee*).
* **Multimedia:** Audio de Pixabay (`main.mp3`) y recursos gráficos ligeros creados en código.

## 🚀 Cómo jugar
1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en tu navegador web favorito (se recomienda Chrome o Firefox).
3. ¡A jugar!

## 📂 Estructura de archivos
* `index.html`: El núcleo del juego (contiene la lógica JS y el estilo CSS).
* `main.mp3`: Banda sonora.
* `icon.png` / `favicon.ico`: Recursos gráficos de identidad del juego.
